# Calculadora de Execução Penal

Aplicativo mobile desenvolvido em Kotlin + Jetpack Compose para auxiliar no cálculo estimado de progressão de regime e benefícios da execução penal, com base na Lei de Execução Penal (LEP) e atualizações do Pacote Anticrime.

---

# 📱 Sobre o Projeto

A Calculadora de Execução Penal foi desenvolvida para facilitar a visualização de datas de progressão de regime, livramento condicional e término da pena de forma prática, rápida e intuitiva.

O sistema realiza cálculos automáticos considerando:

- Tipo do crime
- Reincidência
- Violência ou grave ameaça
- Crime hediondo
- Resultado morte
- Regime inicial
- Remição por trabalho e estudo
- Detração penal
- Data de início da pena
